Code accompanying our submission to Automatica

`L. Baird, M. Cleaveland, A. Schoer and S. Coogan. Signal Temporal Logic Robustness from Copula Bounds and Interval Random Variables. Automatica (in submission), 2026`

Create a new conda environment, and install `pip install -r requirements.txt`

Please run `example_1_uniform.ipynb` to get started. Run `python3 bicycle_paper_experiments.py` to re-generate the figures in the paper.
